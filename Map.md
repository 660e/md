### [Map](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Map)

```javascript
const map = new Map();

map.set("a", 1);
map.set("b", { x: 1 });
map.set("c", [1]);

console.log(map.get("a")); // > 1
console.log(map.get("b")); // > { x: 1 }
console.log(map.get("c")); // > [1]
```
